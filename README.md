# Introdução

Este repositório foi criado como um **breve espaço de reforço de aprendizagem em Java**, com foco em **algoritmos e estruturas de dados**.  
Ele reúne exercícios práticos e pequenos exemplos que ajudam a consolidar conceitos fundamentais da programação e da lógica computacional.
